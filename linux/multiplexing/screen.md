# start a session
* `screen`

# Detatch from current session
* `Ctrl-A D`

# List sessions
* `screen -ls`

# Attach to a session
* `screen -r nnnn` number is what seen from `-ls` output

# Split screen (into focuses)
* `Ctrl-A |`
* `Ctrl-A S`

# Switch between focuses
* `Ctrl-A Tab`

# Start a shell in a focus
* `Ctrl-A c`

# Exit a screen (all its focuses)
* `Ctrl-A \`
