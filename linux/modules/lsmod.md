# List
- `lsmod`
- or `cat /proc/modules`
- lists all modules installed on your system and their dependencies

# Example output
```
Module                  Size  Used by
kvm_intel             138528  0
kvm                   421021  1 kvm_intel
```
# Output format
* Module: module name
* Size: amount of RAM used by the module, in bytes
* Used by: used by what modules
